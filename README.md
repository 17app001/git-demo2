> git reset

- git reset --hard commit-object
	- 名義上後面新增/修改的檔案都會被刪除
- git reset --mixed commit-object
	-　A->U
- git reset --soft commit-object
	- 恢復在暫存區

- git reflog
	- git reset (checkout) commit-object
